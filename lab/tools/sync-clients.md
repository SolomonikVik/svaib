# sync-clients

Папка clients/ не хранится в git (конфиденциальные данные). Она синхронизируется между iMac и MacBook отдельной командой.

## Как синхронизировать

### Шаг 1: Попробуй сам через Bash

**Скачать свежее с iMac на MacBook:**
```
rsync -avz viktorsolomonik@192.168.31.93:~/Projects/svaib/clients/ ~/Projects/svaib/clients/
```

**Отправить изменения с MacBook на iMac:**
```
rsync -avz ~/Projects/svaib/clients/ viktorsolomonik@192.168.31.93:~/Projects/svaib/clients/
```

Пароль не спросит — SSH-ключ настроен.

### Шаг 2: Если команда не сработала (ошибка "No route to host")

Это известный баг macOS — Claude Code не может подключаться к устройствам в локальной сети. Команду нужно запустить вручную.

Скажи Виктору дословно:

> Виктор, открой Terminal на MacBook и вставь эту команду:
>
> (вставить нужную команду из шага 1)
>
> Она скопирует/отправит файлы клиентов. Просто вставь и нажми Enter, больше ничего не нужно.

## Когда синхронизировать

- **В начале работы на MacBook** — скачать свежее с iMac (первая команда)
- **После изменений в clients/ на MacBook** — отправить обратно на iMac (вторая команда)
