# Закрытие сессии knowledge/

## Когда

Виктор говорит "закрываем", "коммить", "заканчиваем" — или ты видишь что работа с knowledge/ завершена.

## Коммит

Закоммитить все изменения в knowledge/ (и только их). Сообщение — человекочитаемое, 1-2 предложения: что появилось / что узнали. Без списков файлов, без технических деталей.

Формат:
```
knowledge: <суть в 1-2 предложениях>
```

Пример:
```
knowledge: новая тема "тестирование AI-кода" — failure modes, TDD+AI, mutation testing, инструменты
```

Плохо: `knowledge: testing.md + UPD !coding.md v17 + cross-refs` — из этого не собрать пост.

**Не пушить.** Только коммит.

## Пуш + сводка для Telegram

Когда Виктор говорит "пушим" или "собери сводку":

1. `git log --oneline @{push}..HEAD -- knowledge/` — собрать все коммиты с последнего пуша
2. Из коммит-сообщений сгенерировать пост для Telegram: краткий, на русском, для AI-сообщества
3. Показать Виктору на утверждение
4. После ОК — `git push`
