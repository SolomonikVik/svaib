# Agents — автоматизация

## Запланированные агенты

### Dispatcher (диспетчер)

Раскидывает результаты чата/транскрибации по файлам management/:

- **Запуск:** хук при закрытии сессии или ручной вызов
- **Анализирует** контент и распределяет:
  - Задачи → ideas
  - Размышления → reflections
  - Решения → decisions
  - Идеи на неделю → plan

## Статус

Скелет. Реализация — после обкатки skills.
