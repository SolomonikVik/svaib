# Plugin — деплоимый пакет Second AI Brain

## Кратко

Claude Plugin в формате Claude Code / Cowork. То, что клиент устанавливает и получает работающую автоматизацию поверх scaffold-структуры.

**Статус:** Скелет. Наполнение — после обкатки методологии на первых клиентах.

## Компоненты

| Что | Папка | Что делает |
|-----|-------|-----------|
| Skills | `skills/` | Методология в исполняемом формате: обработка встреч, ревью, планирование |
| Agents | `agents/` | Автоматизация: диспетчер задач, обновление документации |
| Hooks | `hooks/` | Триггеры: после встречи, при закрытии сессии |

## Связь с моделью подписки

Плагин не хранит данные клиента. Данные — у клиента (в scaffold-структуре). Плагин — интеллектуальный слой поверх.
