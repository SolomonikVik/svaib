# Модель продукта Second AI Brain

## Кратко

Модель продукта svaib для презентации клиентам. Second AI Brain — персональная AI-инфраструктура для руководителя: база знаний + навыки + агенты. Три слоя: данные (Markdown + онтология) → память (Memory, графы, RAG) → интерфейс (чат → команда → скилл → агент). Универсальность: любой AI-инструмент, любая модель, данные у клиента.

## Три компонента продукта

| Компонент | Что это | Пример |
|-----------|---------|--------|
| **Wiki** (база знаний) | Структурированная, обновляемая документация | Миссия, стратегия, история решений, транскрипции встреч |
| **Skills** (навыки) | Инструкции и шаблоны под конкретную компанию | Как писать отчёты, готовить презентации по стандартам ЭТОЙ компании |
| **SubAgents** (агенты) | Автоматизация рутины | Обработка встреч → обновление документации, напоминания о задачах |

## Три слоя архитектуры

### Слой 1: ДАННЫЕ (фундамент)

```
Markdown-файлы + Онтология
├── Ценности / Цели
├── Проекты
│   ├── Цели проекта
│   ├── Встречи
│   └── Задачи
├── Люди / Команда
└── ...
```

**Ключевое:** Структура, а не хаос. Понятно, где что лежит.

### Слой 2: ПАМЯТЬ (три механизма)

**Memory** — правила игры, всегда в голове: как анализируем, какие параметры важны.

**Графы** — знает, что актуально сейчас, а что уже отменили или изменили.

**RAG** — найдёт любой документ, но не знает, в силе он ещё или нет.

> Memory говорит "как думать", Графы — "что сейчас правда", RAG — "где это лежит"

### Слой 3: ИНТЕРФЕЙС (способы взаимодействия)

**Чат** — просто поболтать, спросить что угодно.

**Команда/Промпт** — конкретный запрос, запускаешь руками.

**Скилл** — упакованная экспертиза, знает как делать правильно.

**Агент** — работает сам, без команды.

> Чат — спросил. Команда — запустил. Скилл — по стандарту. Агент — сам работает.

### Обёртка: УНИВЕРСАЛЬНОСТЬ

- Любой интерфейс (Cowork, VS Code, Cogiri...)
- Любая модель (Claude, GPT, DeepSeek...)
- Данные у клиента, не привязан к вендору

## Фреймворк "3 уровня работы с AI"

Показываем клиенту где он сейчас и куда двигаемся:

1. **Чаты** — общаешься, иногда кидаешь файл, каждый чат с чистого листа
2. **Файловая система + проекты** — структура файлов, AI получает контекст
3. **Агенты + интеграция** — субагенты под задачи, работа с файлами напрямую

## Модель подписки "Плагин"

```
КЛИЕНТ                              SVAIB (подписка)
──────                              ────────────────
• Папка с документами               • Skills (методология)
• Встречи, контекст                 • Agents (автоматизация)
• Любой AI-интерфейс                • Онтология (структура)
  (Claude, GPT, Cowork...)          • Обновления, поддержка
```

**За что платят:** Не за хостинг данных и не за доступ к AI, а за интеллектуальный слой — упакованная управленческая экспертиза.
