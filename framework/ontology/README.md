# Ontology — какие сущности существуют

Справочник сущностей Second AI Brain: типы файлов, их поля, связи, правила размещения.

**Границы:** Протоколы и decision frames → `../methodology/`. Готовые шаблоны → `../scaffold/`.

## Файлы

- [ontology.md](ontology.md) — полная спецификация: принципы, большая рамка, 10 сущностей, связи, вложенные проекты
- [entities.md](entities.md) — атомарные сущности (решение, задача, идея и др.)
- [rituals.md](rituals.md) — каталог ритуалов (повторяющиеся управленческие процессы)
- [file_spec.md](file_spec.md) — спецификация файла: как устроена каждая единица контента (YAML, секции, связи, ограничения)
