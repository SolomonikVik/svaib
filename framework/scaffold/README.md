# {Название проекта} — рабочее пространство CEO

Второй мозг руководителя для управления бизнесом. Здесь живёт всё, с чем CEO регулярно работает: личное (ценности, решения, мысли), стратегия компании, команда, оперативка по подразделениям. Агент читает эту папку, чтобы помогать CEO думать, планировать, готовиться к встречам, обновлять файлы по итогам работы.

## Базовая модель — пять слоёв

Пять обязательных папок отвечают на пять разных вопросов:

| Папка | Вопрос |
|-------|--------|
| `01_ceo/` | Кто CEO как человек? |
| `02_strategy/` | Куда идёт компания? |
| `03_team/` | Кто в команде? |
| `04_company/` | Как CEO руководит подразделениями? |
| `05_metrics/` | Как понять, что движемся в нужную сторону? |

Порядок (01 → 05) не случайный: личность руководителя → курс компании → люди в команде → операционное управление → проверка по фактам. Каждый следующий слой опирается на предыдущие. Агент при старте сессии читает в этом же порядке.

Опциональные папки добавляются, когда у CEO появляется живая работа в соответствующем домене — не «на всякий случай»:

- `product/` — стратегия и описания продукта компании
- `clients/` — досье именованных B2B-клиентов (если CEO лично ведёт)
- `projects/` — именованные проекты с горизонтом и завершением (миграция CRM, запуск, стройка)
- `knowledge/` — база знаний CEO (если есть что структурировать)
- `processes/` — регламенты и SOP (если есть формализованные процессы)
- `_inbox/` — входящее на разбор

## Протокол чтения при старте сессии

По умолчанию каждый сеанс:

README (этот) → CLAUDE.md → 01_ceo/01_my_profile.md → 02_strategy/01_vision.md → 02_strategy/02_goal.md

Эти пять файлов дают базовый контекст: кто клиент и куда идёт компания. Остальные стратегические файлы читаются по триггеру задачи, если они уже развёрнуты.

| Триггер | Что подгружать |
|---------|----------------|
| Обработка встречи | `04_company/glossary.md`, файлы затронутого подразделения, профайлы участников из `03_team/` |
| Планирование недели | `04_company/*/02_active.md` всех подразделений, `02_strategy/02_goal.md` |
| Стратегическое обсуждение | `02_strategy/06_decisions.md`, `02_strategy/05_timeline.md` |
| Обсуждение человека | `03_team/{person}.md`, файлы подразделения, где он работает |
| Личная рефлексия CEO | `01_ceo/03_thoughts.md`, `01_ceo/04_decisions.md` |
