# extractors/ — per-client extractors

Python-скрипты, которые извлекают значения метрик из источников клиента (`../source/`) по зафиксированной раскладке координат. Один источник — один extractor.

- Наша территория: код курируем мы, не клиент.
- Создаётся при онбординге под конкретный источник; у каждого клиента свой.
- Возвращает сырые значения с метаданными, производных не считает.

Как строится extractor, контракт, патологии источников — [`../../../methodology/metrics/extractor.md`](../../../methodology/metrics/extractor.md).
