# Базовые метрики бизнеса — {компания}

Базовый файл с ключевыми бизнес-метриками компании — теми, по которым CEO отслеживает движение бизнеса в целом.

## Источник

**Источник:** `{file}.xlsx`, лист «{sheet}», колонка с каноническим именем метрики: «{имя колонки — Метрика / KPI / другое}».

В строках этой колонки стоит каноническое имя метрики из настоящего файла. Помощник работает только со строками, где это имя заполнено — если оно не указано, метрики для помощника не существует.

**Если источников несколько** — заменить эту секцию на `## Источники` с подразделами `### {source_id}`, и в каждой метрике добавить поле `**Источник:** {source_id}`:

```md
## Источники

### source_finance

**Источник:** `finance.xlsx`, лист «P&L», колонка с каноническим именем метрики: «Метрика».

### source_sales

**Источник:** `sales.xlsx`, лист «Pipeline», колонка с каноническим именем метрики: «KPI».
```

## Как заполнять метрики

**Имя метрики** (заголовок `### {имя}`) — каноническое: как метрика реально называется внутри компании. Краткое, однозначное, стабильное (меняется формула — имя не меняется; меняется управленческая сущность — это уже другая метрика).

**Поля паспорта:**
- **Бизнес-смысл** — что метрика значит для бизнеса.
- **Как считается** — правило расчёта с явной временной логикой и способом вычисления (за период / на момент / доля чего-то от чего-то).
- **Единица** — `млн руб` / `%` / `шт` / …
- **Направление** — `рост — хорошо` / `рост — плохо` / `нейтрально`.
- **Источник** — `{source_id}` из секции `## Источники`. Указывается, только если источников в файле несколько.

**Раскладка** — два взаимоисключающих варианта:
- **С группами.** Серия `## {имя группы}` (Продажи / Клиенты / Финэффективность), внутри каждой `### {имя}`. Контейнер `## Метрики` в этом варианте НЕ ставится.
- **Без групп.** Один контейнер `## Метрики`, внутри подряд `### {имя}`. Используется, когда естественных блоков метрик у CEO нет.

В шаблоне ниже — вариант с группами. Если естественных блоков нет — заменить серию `## {имя группы}` на один контейнер `## Метрики`.

## {Имя группы — например, Продажи}

### {Каноническое имя метрики}

**Бизнес-смысл:** {что метрика значит для бизнеса}

**Единица:** {млн руб / % / шт}
**Направление:** {рост — хорошо / рост — плохо / нейтрально}
