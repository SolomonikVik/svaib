# Решения: {направление}

## Кратко

Реестр стратегических решений по направлению с контекстом: что решили, почему, какие альтернативы отвергли. Накопленная логика выбора — через 3 месяца отвечает на "а почему так?". Не журнал встреч, не список задач.

**Работа с файлом:** decision — дописывать при принятии значимых решений по направлению. Растёт медленно. Что сюда: решения, которые меняют стратегию, процессы или структуру направления. Формат: что решили + почему + альтернативы + статус. Что НЕ сюда: company-wide решения (→ ../../02_strategy/06_decisions.md), операционные выборы внутри недели (→ остаются в meetings/), задачи (→ 02_active). Тест: "через 3 месяца кто-то спросит 'почему так?' — это решение будет ответом?" Да → сюда. Нет → meetings/.

**Развёртывание:** создаётся вместе с направлением. Пока решений нет — файл пуст, это нормально. Решения company-wide уровня → ../../02_strategy/06_decisions.md.
