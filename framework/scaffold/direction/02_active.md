# Актуальное: {направление}

## Кратко

Операционный хаб направления: всё, что требует внимания сейчас, в одном месте. Проблемы, задачи, открытые вопросы, риски, идеи и повестка следующей встречи. Единая точка входа для "что горит?" и "что дальше?". State-файл: перезаписывается, не растёт. Вся история — в протоколах meetings/.

**Навигация:** 01_overview.md (что за направление, куда идём) → **02_active.md (что горит сейчас)** → 03_progress.md (что сделали)

**Работа с файлом:** plan — обновлять после каждой встречи и по мере выполнения. Целевой размер: 40-80 строк.

**Workflow выполненных задач:**
1. Задача выполнена → отметить `[x]`, НЕ удалять
2. Перенести в "Следующая встреча" → "Подтвердить: ..."
3. На встрече подтверждено → удалить из файла (история в протоколе meetings/)

**Что сюда:** проблемы, задачи (по owner'ам), открытые вопросы, риски, идеи/гипотезы, повестка следующей встречи. **Что НЕ сюда:** факты о направлении (→ overview), решения с контекстом (→ decisions), хроника (→ progress), коучинговые наблюдения о людях (→ team/{person}.md).

## Связанные файлы

- 01_overview.md — обзор направления (миссия, цели, KPI)
- 04_decisions.md — решения (задача может порождать решение)
- 03_progress.md — хроника (задачи привязаны к встречам)
- meetings/ — протоколы (источник задач, архив выполненных)
- [../team/](../team/) — профайлы лидера и участников
