# Product — всё о продукте

Рабочее пространство продукта: стратегия, описания, коммерция, техническая документация.

**Границы:** Регулярный менеджмент (цели, план, прогресс) → `../management/`. Бизнес-процессы → `../processes/`. Персональная база знаний → `../knowledge/`.

## Что здесь

- **Стратегия** — vision продукта, позиционирование, целевая аудитория
- **Описания** — что продаём, для кого, ключевые ценности
- **Коммерция** — прайсы, офферы, тарифы, карточки продукта
- **Техническое** — спецификации, API, техническая документация

## Как пользоваться

- Создавай файлы по мере появления продуктовых артефактов
- Группируй по подпапкам при разрастании (strategy/, specs/, pricing/)
- Устаревшие версии → `../archive/`
