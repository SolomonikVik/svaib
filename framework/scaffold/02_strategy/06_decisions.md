# Decisions

## Кратко

Реестр принятых решений проекта. Выборы, которые определяют траекторию: закрывают альтернативы, задают направление. Без этого файла LLM будет предлагать уже отвергнутые варианты, а человек забудет почему принял решение. Каждое решение: дата, статус, что решили, контекст, альтернативы, следствия.

**Работа с файлом:** дописывать по мере принятия значимых решений. Растёт медленно. LLM читает при стратегических обсуждениях и пересмотре целей. Решение = выбор, закрывающий альтернативы ("Не идём в B2C", "Выбрали X вместо Y"). НЕ решение = операционный выбор внутри недели. При накоплении >30-40 решений — архивные (статус "Пересмотрено" старше года) → archive/decisions_YYYY.md.
