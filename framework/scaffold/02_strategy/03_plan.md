# Plan

## Кратко

Оперативное управление проектом. Что конкретно делаем прямо сейчас: план периода, план недели, план дня. Содержит ТОЛЬКО будущее и настоящее — прошлое уходит в weekly_progress. Три уровня планирования: период (из goal) → неделя → день. Если есть направления/подпроекты — задачи группируются по ним.

**Работа с файлом:** обновлять ежедневно (план дня), еженедельно по понедельникам (план недели), в начале каждого периода (план периода). Размер компактный и стабильный — не растёт, обновляется. Выполненные задачи → итоги в 04_weekly_progress. Незавершённые задачи → перенести на следующую неделю. LLM читает часто — оперативный контекст.

## Связанные файлы

- [02_goal.md](02_goal.md) — цели и метрики периода, из которых вытекает план периода (depends_on)
- [04_weekly_progress.md](04_weekly_progress.md) — итоги недели переносятся из plan в weekly_progress
- [06_decisions.md](06_decisions.md) — решения влияют на задачи и приоритеты
