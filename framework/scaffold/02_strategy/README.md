# 02_strategy/ — Куда идёт компания

Стратегические артефакты уровня компании: идентичность, цели, план, хроника, история, решения. Всё что здесь — читабельно внешним человеком (инвестор, преемник, новый сотрудник) без дополнительного контекста.

## Что лежит

- `01_vision.md` — кто мы, зачем, идентичность
- `02_goal.md` — куда идём в этом периоде, цели и метрики-план
- `03_plan.md` — что компания делает сейчас (сборка, не state)
- `04_weekly_progress.md` — хроника по неделям
- `05_timeline.md` — сводная история и факт по метрикам
- `06_decisions.md` — стратегические решения компании

Порядок чтения: 01 → 02 → 03. Остальное — по триггеру.

## Правила уровня папки

**Только курс компании.** Личные задачи CEO → `01_ceo/`. Задачи по подразделениям → `04_management/`. Решения уровня одного подразделения → `04_decisions.md` этого подразделения. Если сомнение «курс или подразделение?» — сюда, компания важнее.

**03_plan.md — это сборка.** Задачи живут в `02_active.md` подразделений внутри `04_management/`. План компании собирается из них как срез через призму стратегических целей. Не дублирует задачи, показывает картину целиком. Обновляется понедельничным ритуалом.

**Goal — только план, факт — в timeline.** Целевые значения метрик живут в `02_goal.md`. Фактические — только в `05_timeline.md` по итогам периода. Goal всегда отвечает «куда идём», не «где были».
