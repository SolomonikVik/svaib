# Processes — бизнес-процессы и регламенты

Описания бизнес-процессов, регламенты, SOP (стандартные операционные процедуры).

**Границы:** Протоколы встреч → `../management/meetings/`. Планирование и цели → `../management/`. Продуктовая документация → `../product/`.

## Что здесь

- Бизнес-процессы (описание шагов, ответственных, триггеров)
- Регламенты и политики
- SOP для повторяющихся операций
- Чеклисты и шаблоны процессов

## Как пользоваться

- Описывай процесс когда он повторяется >2 раз
- Один файл — один процесс
- Группируй по подпапкам при разрастании (sales/, hr/, ops/)
- Устаревшие → `../archive/`
