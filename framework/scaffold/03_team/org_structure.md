# Оргструктура

## Кратко

Кто кому подчиняется, какая роль, подразделение, зона ответственности. Чистый документ — можно распечатать и показать команде.

**Работа с файлом:** reference — обновлять при изменениях в команде (приём, увольнение, смена ролей).

```
{CEO}
├── {Имя 1} — {роль} / {подразделение} / {зона ответственности}
├── {Имя 2} — {роль} / {подразделение} / {зона ответственности}
│   ├── {Имя 3} — {роль} / {зона ответственности}
│   └── {Имя 4} — {роль} / {зона ответственности}
└── {Имя 5} — {роль} / {подразделение} / {зона ответственности}
```
