# Задачи: {название проекта}

## Кратко

Актуальные задачи, открытые вопросы, идеи и риски по проекту. Единственное место для "что делать сейчас". State-файл: перезаписывается, не растёт. Вся история — в протоколах meetings/.

**Работа с файлом:** plan — обновлять после каждой встречи и по мере выполнения. Выполненные задачи → удалять (история в meetings/). Решённые вопросы → удалять (решение в decisions или meetings/). Принятые идеи → decisions.md. Целевой размер: 40-80 строк. Что сюда: задачи (по owner'ам), открытые вопросы, идеи/гипотезы, активные риски, повестка следующей встречи. Что НЕ сюда: завершённые задачи (удалить), решения (→ decisions), динамика проекта (→ progress), факты о человеке/бизнесе (→ profile/company).

## Связанные файлы

- project.md — паспорт проекта (что делаем)
- decisions.md — решения (задача может порождать решение)
- progress.md — хроника (задачи привязаны к встречам)
- meetings/ — протоколы (источник задач, архив выполненных)
