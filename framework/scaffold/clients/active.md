# Актуальное: {название проекта}

## Кратко

Операционный хаб проекта: всё, что требует внимания сейчас, в одном месте. Задачи, открытые вопросы, идеи, риски и повестка следующей встречи. Единая точка входа для "где мы?" и "что дальше?". State-файл: перезаписывается, не растёт. Вся история — в протоколах meetings/.

**Навигация:** project.md (что делаем, стейджи) → **active.md (что горит сейчас)** → progress.md (что сделали)

**Работа с файлом:** plan — обновлять после каждой встречи и по мере выполнения. Целевой размер: 40-80 строк.

**Workflow:** задача выполнена → переместить из "Задачи" в "Следующая встреча → Подтвердить" как `- [x] [Owner]: что сделано (дата)`. На встрече подтверждено → удалить.

**Что сюда:** задачи (по owner'ам), открытые вопросы, идеи/гипотезы, активные риски, повестка следующей встречи. **Что НЕ сюда:** решения (→ decisions), динамика проекта (→ progress), факты о человеке/бизнесе (→ profile/company).

## Связанные файлы

- project.md — паспорт проекта (что делаем, стейджи)
- decisions.md — решения (задача может порождать решение)
- progress.md — хроника (задачи привязаны к встречам)
- meetings/ — протоколы (источник задач, архив выполненных)
