# Решения: {название проекта}

## Кратко

Реестр стратегических решений по проекту: что решили, почему, какие альтернативы отвергли. Через 3 месяца отвечает на "а почему так?".

**Работа с файлом:** decision — дописывать при значимых решениях. Что сюда: решения, которые меняют направление проекта. Формат: что + почему + альтернативы + статус. Что НЕ сюда: операционные выборы (→ meetings/), задачи (→ tasks). Тест: "через 3 мес спросят 'почему так?' — это будет ответом?" Да → сюда.

## Связанные файлы

- project.md — паспорт проекта
- tasks.md — задачи (решение может порождать задачу)
