# Knowledge — персональная база знаний

Накопленные знания, которые помогают принимать решения. Не привязана к планированию — это ссылки, заметки, инсайты, инструменты.

**Границы:** Планирование и цели → `../management/`. Входящее на разбор → `../_inbox/`.

## Как пользоваться

- Добавляй ссылки, заметки, находки по мере появления
- Группируй по темам (подпапки)
- Периодически пересматривай — устаревшее → `../archive/`

## Структура

Зависит от специфики проекта. Примеры подпапок:
- `tools/` — инструменты, которые используем
- `methods/` — методики, подходы, фреймворки
- `contacts/` — полезные контакты и ресурсы
