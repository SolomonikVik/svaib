## Протокол чтения

### По умолчанию (каждый сеанс)
```
README → SKILL.md → 01_vision → 02_goal → 03_plan
```

### По триггеру
- Стратегическое обсуждение → 05_timeline, decisions
- Подведение итогов → 04_weekly_progress
- Планирование квартала → ideas, metrics
- Анализ состояния → reflections

## Ритуалы

| Когда | Что | Файлы |
|-------|-----|-------|
| Каждое утро | План дня | 03_plan |
| Каждый понедельник | Итоги недели → новый план | 03_plan → 04_weekly_progress |
| Начало квартала | Цель + KR + план | 02_goal, 03_plan, ideas |
| Конец квартала | Итоги → timeline | 04_weekly_progress → 05_timeline |

## Правила

1. **Дословное копирование.** При переносе заметок — копировать текст как есть. Не перефразировать, не сокращать.
2. **Один файл — одна роль.** Не смешивать содержимое разных файлов.
3. **Факты в goal, итоги в timeline.** KR план — в goal, KR факт — только в timeline.
4. **ISO 8601.** Даты в формате YYYY-MM-DD.
