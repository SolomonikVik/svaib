# Глоссарий

## Кратко

Служебный контекст для AI — помогает правильно читать транскрипты встреч: варианты имён, ошибки транскрибации, жаргон и сокращения компании.

> **Временное решение.** Отдельный файл glossary.md в management/ — быстрое решение для снятия блокера (L1 не знал людей, 80% ошибок). Формат и расположение могут измениться: глоссарий может стать частью org_structure, генерироваться автоматически из встреч, или переехать. Контекст людей и терминов нужен постоянно — но этот конкретный файл не финальная форма.

**Работа с файлом:** reference — обновлять после обработки встреч. Если встретилось новое имя, ошибка транскрибации, незнакомый термин — добавить сюда. Растёт со временем.
