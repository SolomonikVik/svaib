# Meetings — протоколы управленческих встреч

Протоколы (meeting notes) управленческих встреч. Результаты встреч питают недельные итоги (`../04_weekly_progress_YYYY.md`).

**Границы:** Недельные итоги → `../04_weekly_progress_YYYY.md`. Решения из встреч → `../decisions.md`. Бизнес-процессы → `../../processes/`.

## Формат файла

Один файл на встречу: `YYYY-MM-DD_{тема}.md`

Рекомендуемые секции:
- Участники
- Повестка
- Ключевые решения
- Задачи (кто, что, когда)

## Как пользоваться

- Создавай файл после каждой значимой встречи
- Ключевые решения → дублировать в `../decisions.md`
- При подведении недельных итогов — просмотреть за неделю
