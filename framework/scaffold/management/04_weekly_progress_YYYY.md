# Weekly Progress YYYY

## Кратко

Детальный лог проекта: что реально происходило неделя за неделей. "Сырые данные" — из них собираются квартальные и годовые итоги в timeline. Одна запись на неделю: фокус, сделано, не сделано, заметки. Новый файл каждый год (04_weekly_progress_2026.md, 04_weekly_progress_2027.md).

**Работа с файлом:** обновлять каждый понедельник — итоги прошлой недели. Только дописывать новые записи сверху (обратная хронология внутри квартала). Не менять старые записи. Растёт (~50 записей в год). LLM читает по запросу, НЕ по умолчанию. В конце квартала — собрать итоги из этого файла → 05_timeline.

## Связанные файлы

- 03_plan.md — итоги недели берутся из выполненных/невыполненных задач плана
- 05_timeline.md — квартальные итоги собираются из недельных записей (feeds_into)
