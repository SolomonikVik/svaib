# Ideas

## Кратко

Структурированная парковка для идей, гипотез и отложенных задач. Три типа записей: идеи (нужно оформить и обдумать), гипотезы (нужно проверить/валидировать), задачи на потом (можно брать в plan напрямую). Каждая запись с датой. При планировании квартала — просматривать и вытаскивать релевантное в plan.

**Работа с файлом:** дописывать по мере появления, в любой момент. LLM читает при планировании квартала, НЕ по умолчанию. Раз в квартал — чистить: удалить неактуальное. Растёт, периодически чистится. Набросать запись можно не думая о структуре — LLM распределит по секциям.

## Связанные файлы

- 03_plan.md — задачи из ideas берутся в план при планировании квартала (feeds_into)
- 02_goal.md — цели определяют, какие идеи релевантны
- decisions.md — решение может закрыть идею или превратить гипотезу в задачу
- reflections.md — размышления могут порождать идеи и гипотезы
