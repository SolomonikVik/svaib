# Plan

## Кратко

Оперативное управление проектом. Что конкретно делаем прямо сейчас: план квартала, план недели, план дня. Содержит ТОЛЬКО будущее и настоящее — прошлое уходит в weekly_progress. Три уровня планирования: квартал (из goal) → неделя → день. Если есть направления/подпроекты — задачи группируются по ним.

**Работа с файлом:** обновлять ежедневно (план дня), еженедельно по понедельникам (план недели), раз в квартал (план квартала). Размер компактный и стабильный — не растёт, обновляется. Выполненные задачи → итоги в 04_weekly_progress. Незавершённые задачи → перенести на следующую неделю. LLM читает часто — оперативный контекст.

## Связанные файлы

- 02_goal.md — цели квартала и KR, из которых вытекает план квартала (depends_on)
- 04_weekly_progress_YYYY.md — итоги недели переносятся из plan в weekly_progress
- ideas.md — при планировании квартала: просмотреть ideas, вытащить релевантное
- decisions.md — решения влияют на задачи и приоритеты
