# Framework Status

> **Правила работы с этим файлом:**
> - Это файл СОСТОЯНИЯ фреймворка, не документация. Перезаписывается, не растёт.
> - Обновляется в двух случаях: (а) задача по framework ЗАВЕРШЕНА (вся задача, не промежуточный шаг) → добавь строку в Мини-лог, (б) ритуал понедельника → вшей мини-лог в секции, очисти лог. НЕ обновляй после каждого шага или сессии.
> - Мини-лог — единственное что растёт в течение недели. Одна строка на завершённую задачу.
> - Понедельник (ритуал): перечитай мини-лог, обнови секции, очисти мини-лог.
> - Максимум файла: ~100 строк. Если больше — что-то лишнее. Детали живут в файлах, на которые ссылаемся.
> - НЕ дублируй содержание worklog. Worklog = активная задача (одна). Status = состояние направления (весь framework).
