# svaib

AI-мастерская, которая делает предпринимателей сильнее через AI.

## Что это

**svaib** — экспериментальная AI-компания Виктора Соломоника. Строим персональную AI-инфраструктуру для руководителей: не чат-бота, а полноценного цифрового партнёра, который понимает бизнес, людей и контекст.

Три направления:

- **Продукт** — Second AI Brain
- **Лаборатория** — внутренняя AI-мастерская, ускоряющая всё остальное
- **Публикации** — Telegram, семинары, YouTube — открыто делимся движением

Стадия: **100 недель кринжа** — публичный контракт с собой (старт сентябрь 2025). Первая половина — построить рабочий продукт и фреймворк. Вторая — доказать бизнес-состоятельность. Вся история — в открытую.

## Second AI Brain

Персональная AI-инфраструктура для руководителя. Не SaaS, не платформа — методология + набор инструментов, которые разворачиваются поверх данных клиента.

Пять слоёв:

1. **Онтология** — какие сущности есть в бизнесе (люди, проекты, решения, ритуалы)
2. **Методология** — как с ними работать (протоколы встреч, онбординг, анализ)
3. **Scaffold** — каркас папок и файлов, структура рабочего пространства
4. **Skills** — библиотека промптов и навыков по доменам
5. **Plugin** — собранный пакет, который деплоится клиенту

Принцип: данные остаются у клиента, методология — у нас.

## База знаний

`knowledge/` — стратегический актив проекта. AI-технологии меняются еженедельно, а модели отстают на 6-12 месяцев. База компенсирует этот разрыв: 8 категорий (coding, tools, skills, agents, plugins, prompting, context, cases), 50+ файлов с метаданными и статусами обработки.

Два назначения:

1. **RAG для AI** — помощники используют базу как актуальный источник знаний
2. **Публикация** — обработанные материалы пойдут на сайт для сообщества

## Структура репозитория

| Папка        | Назначение                                                              |
| ------------ | ----------------------------------------------------------------------- |
| `framework/` | Ядро продукта: онтология, методология, scaffold, skills, plugin         |
| `knowledge/` | Актуальная память AI-помощника: свежие знания об инструментах и методах |
| `meta/`      | Стратегия: vision, OKR, маркетинг, продуктовый вижн                     |
| `dev/`       | Сайт svaib.com (Next.js, Vercel) + техническая документация             |
| `clients/`   | Ранние клиенты + playbook консалтинга *(локально, не публикуется)*      |
| `lab/`       | AI-лаборатория: принципы, процессы, гайды по созданию помощников        |
| `.claude/`   | Локальная инфраструктура AI-помощников: роли, навыки, автоматизация      |
