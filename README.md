# svaib

AI-мастерская, создающая AI-решения для бизнеса. Миссия: переводчик между сложными AI-технологиями и практическими потребностями бизнеса.

**Текущий продукт:** Second AI Brain — персональная AI-инфраструктура для руководителя. Данные у клиента, методология у нас.

## Структура репозитория

| Папка        | Назначение                                                          |
| ------------ | ------------------------------------------------------------------- |
| `framework/` | Ядро продукта Second AI Brain (онтология, методология, плагин)       |
| `meta/`      | Стратегия проекта и продукта (ЧТО, ЗАЧЕМ, КТО, КОГДА)               |
| `dev/`       | Разработка и реализация (КАК строить продукт)                        |
| `knowledge/` | База внешних знаний об AI-инструментах и методах                     |
| `clients/`   | Клиентские проекты и методология консалтинга                         |
| `lab/`       | AI-лаборатория: принципы, процессы, правила создания помощников       |
