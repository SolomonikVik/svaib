# Tools — AI-инструменты и платформы

## Кратко

Платформы автоматизации и AI-инструменты, НЕ связанные с написанием кода: n8n, Dify, Langflow, базы данных, инфраструктурные сервисы. Это "ЧЕМ автоматизируем процессы". Не путать с: coding/ (среды разработки, AI-кодинг ассистенты — Claude Code, Cursor, Codex) и agents/ (tools в контексте агентов — function calling, MCP).

## Файлы в этой папке

- [cowork.md](cowork.md) — Cowork: агентная платформа Anthropic для knowledge workers (не разработчиков). Plugins = бандлы из Skills + Agents + Commands + MCP. Формат идентичен Claude Code. Прямая связь с продуктом SVAIB
- [web-scraping-apis.md](web-scraping-apis.md) — Web Scraping APIs: прокси-сервисы для обхода anti-bot защиты (scrape.do, Firecrawl, ScraperAPI, Apify). Интеграция с Claude Code через MCP и curl
