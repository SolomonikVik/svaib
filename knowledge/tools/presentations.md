# Презентации с AI

## Кратко

Обзор инструментов для создания презентаций с участием AI. Два полюса: markdown-first (Slidev, Marp) для file-first подхода и визуальные AI-генераторы (Lovable, Gamma, Slider AI) для быстрого результата без навыков дизайна. Ключевой тренд: AI убирает барьер дизайна, контент становится главным.

---

## Два подхода

| Подход | Суть | Для кого | Примеры |
|--------|------|----------|---------|
| **Markdown-first** | Слайды = текстовый файл. Git, версионирование, AI редактирует напрямую | Разработчики, file-first адепты | Slidev, Marp |
| **Визуальный AI** | Закидываешь контент → получаешь готовый дизайн | Менеджеры, предприниматели, все кто не дружит с дизайном | Lovable, Gamma, Slider AI |

---

## Инструменты

### Slidev

Markdown-first инструмент для презентаций. Vue 3 + Vite под капотом.

- **Сайт:** https://sli.dev
- **Ключевое:** Один файл `slides.md` + одна команда `npx slidev`. Всё остальное — опционально
- **Экспорт:** PDF, PPTX, PNG, статический сайт
- **Фичи из коробки:** Presenter View с таймером и remote control, подсветка кода с пошаговым выделением, Magic Move (анимация трансформации кода), Monaco Editor (live coding в слайде), Mermaid-диаграммы, LaTeX, рисование на слайдах, VS Code extension
- **AI-ready:** Официальный Agent Skill (`npx skills add slidevjs/slidev`). Презентацию можно разбить на отдельные .md файлы — удобно для context engineering
- **Почему markdown — мощно:** find & replace / regex для массовых правок, смена темы в одной строке frontmatter, Git-friendly диффы, hackable (npm-пакеты, API, интерактив)
- **Позиционирование:** "Presentation slides for developers", но с AI-агентами доступно шире

Источник: пост @nobilix

### Lovable

Vibe-coding платформа (lovable.dev), которая неожиданно хороша для презентаций.

- **Кейс:** Человек прошёл путь ChatGPT → AI-генераторы → NotebookLM → Claude Code → Lovable. Неделя мучений с другими инструментами, Lovable выдал живую редактируемую презентацию сразу
- **Сильная сторона:** Нормальный дизайн без навыков дизайна. "Без боли, без обучений, без шаманства с промптами"
- **Ограничение:** Основное назначение — сайты и приложения, презентации — побочный юзкейс

### NotebookLM (Google)

AI-инструмент для работы с документами. Загружаешь файлы и ссылки → генерирует контент.

- **Для презентаций:** Выдаёт хороший PDF на основе загруженных материалов
- **Ограничение:** PDF нередактируемый — нельзя подправить результат

### Slider AI

Российский онлайн-редактор презентаций с AI.

- **Сайт:** https://slider-ai.ru
- **Суть:** Шаблоны + AI-генерация + 40+ типов диаграмм
- **Тарифы:** Бесплатный базовый доступ, Pro от 399 ₽/мес, Business от 1195 ₽/мес
- **Аудитория:** Маркетологи, менеджеры, предприниматели

### Другие упоминания

- **Marp** — Markdown-презентации, минималистичный, но ограничен по возможностям (предшественник Slidev в workflow автора)
- **Gamma** — AI-генератор презентаций (визуальный подход)
- **Figma Slides** — Slides от Figma (для дизайнеров)
