# STT-модели для русского языка

## Кратко

Сравнение моделей распознавания русской речи по WER-бенчмаркам (2025-2026). GigaAM-v3 от Сбера доминирует с двукратным отрывом от Whisper: средний WER ~8% vs ~16%. На зашумлённом аудио (дальний микрофон, комната) разрыв ещё больше: 4% vs 18%. Для продукта, транскрибирующего встречи на русском, GigaAM-v3 — первый выбор в обоих сценариях.

---

## Ключевые выводы

1. **Специализированные русские модели обогнали мультиязычные.** GigaAM-v3 (240M параметров, MIT) точнее Whisper large-v3 (1550M) в 2+ раза на русском. Эпоха "Whisper для всего" закончилась.

2. **На зашумлённом аудио разрыв критический.** Golos Farfield (дальний микрофон): GigaAM 4,1% vs Whisper 17,6% — разница в 4,3 раза. Это именно сценарий "диктофон в переговорной".

3. **Whisper галлюцинирует на тишине.** Задокументированная системная проблема: на паузах и не-речевых сегментах Whisper генерирует несуществующий текст. Для встречных записей с паузами — критично. Обязателен VAD-препроцессинг.

4. **Диаризация — отдельная задача.** Ни одна open-source ASR-модель не включает нативную диаризацию. Стандартное решение: pyannote community-1 (language-agnostic, DER ~7% на meeting rooms).

### Бэкенды исполнения (важно для MacWhisper)

Whisper-модели могут исполняться через разные бэкенды с разной производительностью:

- **Whisper C++ (GGML)** — стандартный бэкенд, работает на любом железе (M1, Intel, GPU). Все размеры от Base (150 MB) до Large V3 (3 GB).
- **WhisperKit** — оптимизированный бэкенд для Apple Silicon (M1/M2/M3), использует ANE (Apple Neural Engine). Быстрее Whisper C++ на Apple Silicon. Доступны Small (483 MB) и Large v2 (3.1 GB). **Не работает на Intel Mac.**
- **Parakeet v3** (494 MB) — отдельная архитектура NVIDIA, **только Apple Silicon**. Экстремальная скорость (300x RT), но нет публичных бенчмарков WER для русского.

На Intel Mac из локальных вариантов работает только Whisper C++ (и то медленно — терпимо лишь Small/Base). Решение для Intel — облачные API.

## Рекомендации по сценариям

### Онлайн-встречи (чистый звук)

**GigaAM-v3 e2e_rnnt.** Чистое аудио — сильная сторона: CommonVoice 2,6%, аудиокниги 3,4%. Вариант e2e_rnnt выдаёт текст с пунктуацией и нормализацией без дополнительных модулей. 240M параметров, ~2 GB VRAM или CPU через ONNX. Для диаризации — pyannote community-1.

Облачная альтернатива: **ElevenLabs Scribe v2** ($0,22-0,40/час, нативная диаризация, заявленный WER <= 5% на русском).

### Офлайн-встречи (микрофон в комнате, шум, несколько голосов)

**GigaAM-v3 e2e_rnnt + pyannote.** Именно здесь максимальный отрыв: Golos Farfield 4,1% vs Whisper 17,6%. Модель дообучена на колл-центрах, голосовых сообщениях, речи с фоновой музыкой. В отличие от Whisper, не галлюцинирует на паузах (CTC/RNNT-архитектура).

Облачный fallback: **Yandex SpeechKit** ($0,08/час отложенно, SpeechKit Hybrid для on-premise).

### Стриминг (live-транскрипция)

**T-one** (71M параметров, Apache 2.0) — единственная open-source модель с нативным стримингом для русского. Чанки 300мс, задержка ~1с. Оптимизирован под телефонию, на wide-band аудио встреч может быть менее точен. Нет пунктуации — нужен отдельный модуль.

## Известные провалы

- **Whisper**: галлюцинации на тишине, потеря контекста на стыках 30-сек сегментов, WER 17-27% на зашумлённом русском
- **GigaAM-v3**: только русский (на англ. — мусор), `.transcribe()` ограничен 25 сек (нужен `.transcribe_longform()` + pyannote), требует float32 на входе, нет стриминга
- **T-one**: нет пунктуации, нет нормализации, оптимизирован под 8 кГц телефонии, KenLM не работает на Windows
- **NVIDIA Parakeet v3**: катастрофа на телефонии/колл-центре (29-32% WER), русский добавлен недавно — мало пользовательского опыта
- **NVIDIA Canary v2**: худший результат среди всех на русском (средний WER 20,2%, колл-центр 34-36%)

## Облачные API (краткая сводка)

| Провайдер | Цена/час | Диаризация | Публичный WER (рус.) |
|-----------|------:|---|---|
| **Groq API** | Free tier | Нет | Whisper Turbo ~9,5% |
| **ElevenLabs Scribe v2** | $0,22-0,40 | Нативная, до 32 спикеров | <= 5% (заявл.) |
| **Soniox API** | $6-12/мес | Нативная code-switching | 6,2% |
| **Yandex SpeechKit** | $0,08-0,30 | Нативная, до 2 дикторов | Не опубликован |
| **AssemblyAI Universal-2** | $0,15 | +$0,02/час | <= 10% (заявл.), только batch |
| **Deepgram Nova-3** | $0,46-0,55 | +$0,12/час | Не опубликован |
| **Google Cloud STT** | $0,18-0,96 | Chirp 3 | Не опубликован |

**Groq API** — единственное реальное решение для real-time транскрибации (216x RT на Whisper Turbo). Бесплатный тир покрывает большинство задач. Единственный вариант для Intel Mac, где локальные модели непрактичны. Для повседневной диктовки на русском Whisper Turbo через Groq — проверенный надёжный выбор.

**Soniox API** — лучший облачный WER для русского (6,2%), единственный сервис с нативной поддержкой code-switching (русский + английский в одном потоке). Малоизвестен, подключение через API.
