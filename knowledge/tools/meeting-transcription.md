# Запись и транскрибация встреч

## Кратко

Практический гайд по записи рабочих встреч на русском языке. Ни один инструмент не закрывает все сценарии — но комбинация из двух (софт + аппаратный рекордер) покрывает 95% задач. Ключевой факт: качество русской транскрибации у западных инструментов в 2-5 раз хуже английской (WER Whisper ~16% vs ~3%), поэтому возможность подключить внешнюю модель или пост-обработку критична. Качество STT-моделей — в [russian-stt-models.md](russian-stt-models.md).

## Дерево решений

### Онлайн-встречи на десктопе

**Mac/Win + Zoom/Meet/Teams, без ботов** — **Granola** ($14/мес). Захватывает системное аудио невидимо, поддерживает русский, MCP-сервер и Zapier-вебхук с полным транскриптом в Markdown. Альтернатива если команда в Notion: **Notion AI Meeting Notes** ($20/мес Business) — транскрипт сразу как Notion-страница с нативным экспортом в Markdown.

**Боты допустимы, максимум интеграций** — **Fireflies.ai** ($10-29/мес). 100+ интеграций, GraphQL API на всех планах, вебхуки, MCP-сервер. Русский с автодетектом. Бот видим как участник.

**Windows + Teams** — **Krisp** ($8/мес годовой). Виртуальное аудиоустройство на уровне ОС, записывает любой звонок без бота + шумодав. Русский на Pro-плане. Или **Fellow 5.0** ($7/мес) — полная система управления встречами с MCP-сервером.

**Смешанная инфра (Mac + Win + мобильные), один инструмент** — **Circleback** ($21/мес). Десктоп на обеих ОС, мобильное для офлайн, самый широкий MCP-сервер (встречи + календарь + email). Бот/botless, 100+ языков.

**Российская инфра (Telemost, Битрикс24, amoCRM)** — **mymeet.ai**. Единственный инструмент под российский рынок: данные в РФ (ФЗ-152), нативная интеграция с Битрикс24 и amoCRM.

### Нативные транскрипты платформ

| Платформа | Русский | Формат | Примечание |
|-----------|---------|--------|------------|
| **Microsoft Teams** | Да | DOCX/VTT, Microsoft Graph | Часто лучший вариант при M365 |
| **Zoom** | Да | VTT (cloud recording) | Только при включённом cloud recording |
| **Контур Толк** | Да | API по ключу | Протокол с задачами и ответственными; F5AI как готовый коннектор |
| **Google Meet** | **Нет** | — | Русского нет на апрель 2026 |

### Офлайн-встречи в переговорной

**Ноутбук с собой** — **MacWhisper Pro** ($29 разово, Mac) или **Vibe** (бесплатно, кроссплатформенный). Запись с микрофона, транскрипт в Markdown. Полностью локально. Для лучшего русского в Buzz можно загрузить fine-tuned модель (WER ~6,4% вместо 10%).

**Ноутбука нет** — **Plaud Note Pro** ($189). Карточка размером с кредитку, запись до 50 часов, автотранскрибация на русском через приложение. Экспорт в TXT/DOCX/MD (27+ форматов). Для больших залов (5+ метров) — лучший микрофон среди аппаратных рекордеров (4 MEMS + VPU).

### Встречи с телефона (iPhone/Android)

**Zoom или Контур Толк с телефона** — если встреча идёт через платформу, запись и транскрипт работают на стороне платформы, не на устройстве. На iPhone это единственный надёжный способ для прогулочных звонков — никакие сторонние приложения не записывают системный аудио. Настраивается один раз, дальше транскрипты приходят сами.

### Телефонные звонки (не через платформу)

**iPhone, с уведомлением** — iOS 18.1 встроенная запись (смена региона на US). Русский поддержан для записи, но Call Transcription русский НЕ включает — нужен второй шаг через MacWhisper/Aiko.

**iPhone, тихая запись** — **Plaud Note / Note Pro** ($159-189) на MagSafe. Вибрационный сенсор (VCS) записывает обе стороны без динамика и уведомления. Работает с любым типом звонка: сотовый, WhatsApp, Telegram, FaceTime. **Единственный надёжный способ тихой записи звонков на iPhone в 2026.**

**Android** — встроенная запись Samsung/Xiaomi (бесплатно) или Cube ACR. Файлы в папке Call → загрузить в Whisper для русской транскрибации.

### На ходу — прогулка, разговор

**Носимый рекордер** — **Plaud NotePin S** ($179). Клипса на лацкан, 20 часов батареи. Но НЕ записывает звонки (нет VCS) — только микрофон.

**На ходу + звонки** — **Plaud Note Pro** ($189) на MagSafe. Записывает и звонок (VCS), и разговор вокруг (микрофоны).

### Real-time транскрибация

**Groq API + Whisper Turbo** — единственное реальное решение для live-транскрибации (216x RT, практически нулевая задержка). Бесплатный тир покрывает большинство задач. Для повседневной диктовки на русском — проверенный надёжный выбор. Единственный вариант для Intel Mac. Подключение в MacWhisper: Settings → Cloud Services → API key.

### Максимальная приватность

**Ни один байт в облако** — MacWhisper Pro (Mac) или Vibe (Win/Linux) для записи + локальный Whisper. Для AI-саммари — **Char (ex-Hyprnote)** с локальным LLM через Ollama.

**Лучшая русская модель локально** — **Buzz** + fine-tuned `antony66/whisper-large-v3-russian` (WER 6,4%). Для диаризации — WhisperX с pyannote. Или **GigaAM-v3** (абсолютный лидер, WER ~8% средний) — детали в [russian-stt-models.md](russian-stt-models.md).

## Автоматизация транскриптов

Во всех трёх основных сценариях транскрипт может приходить автоматически, без ручных действий.

**Zoom** — Auto-record в настройках → вебхук "запись готова" → скрипт скачивает транскрипт (VTT) и аудио через API. Если качество русского не устроит — аудио прогоняется через Whisper/GigaAM.

**Контур Толк** — API есть, авторизация по ключу. Готовое решение: **F5AI** (f5ai.ru) подключается к Толку и вытягивает записи в течение 45 мин после встречи. Можно использовать F5AI или написать свой скрипт.

**Plaud Note** — встроенная функция AutoFlow: при синхронизации автоматически делает транскрипт и шлёт на email. Zapier-интеграция с триггером "Transcript Ready" — автовыгрузка в Google Drive, Notion, Slack. Без разработки, за час.

**MacWhisper (наш пайплайн)** — настроен и работает. Скилл `macwhisper-transcript` автоматически подтягивает транскрипт из локальной SQLite-базы MacWhisper в Claude Code. Команда: `bash scripts/macwhisper_transcript.sh --list` → выбор сессии → экспорт в `_transcript.md` клиента.

### MCP-серверы (2026)

Наиболее перспективный интеграционный путь. Официальные MCP-серверы: **Granola** (с фев 2026), **Fireflies**, **Fellow**, **Circleback** (самый широкий: встречи + календарь + почта), **tl;dv**, **Otter.ai**. Deepgram CLI тоже включает MCP-сервер для транскрибации через API.

## Чего избегать

**Otter.ai** — только английский, французский, испанский, японский. Русского нет. Плюс федеральный иск (2025) за запись без согласия.

**Google Meet нативные транскрипты** — русского нет (апрель 2026). Коварная ловушка: captions развиваются, но транскрипты как артефакт на русском не заявлены.

**Limitless Pendant** — приобретён Meta (декабрь 2025), продажи прекращены, поддержка только до конца 2026. Сервис отключён в ЕС и Великобритании.

**Ловушка "100+ языков"** — почти все западные инструменты заявляют 100+ языков, но это значит лишь использование мультиязычного Whisper. Заявленная поддержка русского ≠ хорошее качество русского.

## Быстрый чеклист

- **Онлайн, Mac, без ботов** → Granola или Notion AI Meeting Notes
- **Онлайн, Windows** → Krisp (Pro) или Fellow
- **Онлайн, боты ОК, макс интеграций** → Fireflies
- **Российская инфра** → mymeet.ai
- **Офлайн с ноутбуком** → MacWhisper Pro (Mac) или Vibe
- **Офлайн без ноутбука** → Plaud Note Pro
- **Встречи с телефона (Zoom/Толк)** → облачная запись платформы, на iPhone это единственный путь для прогулок
- **Звонки iPhone (не через платформу)** → Plaud Note Pro (тихо) или iOS 18.1 (с уведомлением)
- **Звонки Android** → встроенная Samsung/Xiaomi или Cube ACR
- **На ходу** → Plaud NotePin S (клип) или Note Pro (MagSafe)
- **Real-time** → Groq API + Whisper Turbo (единственный вариант с нулевой задержкой)
- **Диктовка (русский)** → Whisper Turbo через Groq (надёжно, $0)
- **Приватность** → MacWhisper Pro / Vibe + Char для AI-саммари
- **Один инструмент на всё** → Circleback + Plaud Note Pro
