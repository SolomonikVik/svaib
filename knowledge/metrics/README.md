# metrics/ — LLM-ассистент с метриками

Знания об архитектуре встраивания бизнес-метрик и числовых данных в LLM-ассистента руководителя: слои (semantic, snapshot, verified queries), форматы данных в контексте LLM, что считать в коде vs LLM, MCP-каналы доступа к данным.

**Границы:** не про общую BI или dashboarding. Только про то, как данные доходят до LLM и обратно. Промптинг как техника → [../prompting/](../prompting/), агентные системы и MCP вообще → [../agents/](../agents/), Context Engineering и RAG в общем → [../context/](../context/).
