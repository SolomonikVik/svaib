# Dev — обзор направления

## Кратко

Dev — разработка и реализация: публичный сайт svaib.com, техническая инфраструктура, архивные product docs. Отвечает на вопрос **КАК строить**. Два основных артефакта: код сайта (`src/`, Next.js 16) и техническая документация (`dev_context/`). Стратегия и позиционирование продукта живут в `meta/` и `framework/` — здесь только реализация.

**Работа с файлом:** reference — обновлять при смене фокуса направления или стадии (новый фреймворк сайта, новый крупный технический слой). Не обновлять при ежедневной работе (текущие задачи → `02_active.md`, сделанное → `03_progress.md`).

## Миссия

Dev превращает стратегию и фреймворк в **работающий публичный интерфейс svaib** и поддерживает техническую инфраструктуру. Сайт svaib.com — витрина продукта Second AI Brain и воронка привлечения клиентов. `dev_context/` — канон технических решений для любой разработки.

**Входит в scope:** код сайта (`src/`, Next.js/React/Tailwind, Vercel), дизайн-система (синхронизирована с брендбуком marketing/), техническая документация (`dev_context/`), инфраструктура (серверы, домены, API), архивные workflows (n8n), интеграции (прежде — Supabase, Dify; сейчас — Vercel, Claude API).
**НЕ входит:** стратегия продукта (→ `meta/management/`), позиционирование и бренд (→ `meta/marketing/`), ядро продукта Second AI Brain (→ `framework/`), AI-помощники для внутренней работы (→ `lab/` → `.claude/`).

---

## Рамка

**Стадия:** сайт на Next.js 16 live, деплой через Vercel. Пивот marketing/бренд v2 в проработке — главная страница переведена, остальные разделы пока на старой системе. Архивные product docs в `dev_context/` большей частью устарели (записи декабря 2025, до пивота на Second AI Brain) — требуют ревизии.

**Tech Stack:** Next.js 16 + React 19 + Tailwind CSS. Деплой: Vercel (auto-deploy on push to main). Публичные маршруты: `/`, `/second-ai-brain`, `/lab`, `/knowledge`, `/archive`. Внутренний: `/vote` (выключен через env).

## Цель декады (недели 18→30)

Dev играет **сервисную роль** — поддерживает текущий сайт, чинит проблемы, синхронизируется с брендбуком v2 по мере его утверждения. Собственных крупных инициатив на декаду нет.

Текущие приоритеты (см. [02_backlog.md](02_backlog.md)):
- Сайт должен открываться в России (есть гипотеза решения)
- Ревизия `dev_context/` — выявить актуальное, архивировать устаревшее

Детали целей svaib в целом → [../meta/management/02_goal.md](../meta/management/02_goal.md).
