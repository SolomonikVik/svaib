# _inbox/

> Входящие файлы на разбор и форматирование.

## Назначение

Временное хранилище для черновиков и файлов, которые нужно:
- Отформатировать по правилам контекста
- Разобрать и переместить в нужную папку (meta/dev/pub)

## Правило

Папка должна стремиться к пустоте. Кинул → разобрали → файл переехал.

## Как использовать

1. Кинь файл сюда
2. Запусти `/svaib-context`
3. Скажи "разбери inbox" или укажи конкретный файл
