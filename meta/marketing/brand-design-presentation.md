## Цветовые роли на слайдах

Палитра из [brand.md](brand.md), но специфика применения:

| Цвет | Роль на слайдах |
|------|-----------------|
| Dark Teal `#008B7F` | Заголовки H1, teal stripe, структурные акценты |
| Dark Blue `#2C3E50` | H2, body text, подзаголовки |
| Pink `#FF4D8D` | Только акценты: слово "AI", ключевые цифры |
| Bright Teal `#00B4A6` | Иконки, иллюстрации, декоративные элементы |
| Yellow `#FFD600` | Иллюстрации |
| Gray `#6B7280` | Footnotes, captions, мелкий текст, footer |

**Фоны слайдов:**
- `#FFFFFF` — основной (по умолчанию)
- `#F0FDFB` — для разделительных слайдов (светлый бирюзовый)
- `#E0F7F5` — заливка карточек/фигур (изредка)

**Запрещено:** тяжёлые заливки, градиенты, розовый как фон. Максимум 2 цветовых акцента на слайд.

## Композиция

### Пропорции текст ↔ визуал

- **50/50** — равное деление
- **60/40** — акцент на картинке (или наоборот)

Каждый элемент центрируется **в своей зоне**, не по всему слайду.

### Отступы

| Параметр | Значение |
|----------|----------|
| От края слайда | 2-2.5 см |
| Заголовок → контент | 1-1.5 см |
| Картинка ↔ текст (разрыв) | 1.5-2 см |
| Между текстовыми блоками | 0.8-1 см |

### Вертикальное размещение

**Align Top (по умолчанию):** картинка и текст начинаются сверху своих зон. Для много текста, чёткой структуры.

**Align Middle (для вау):** контент по центру вертикально. Для 1-2 пунктов, когда нужен воздух.

## Лейауты

Чередуй лейауты, не повторяй один паттерн:

| Паттерн | Когда |
|---------|-------|
| Two-column (текст + визуал) | По умолчанию для контента с иллюстрацией |
| Icon + text rows | Списки фич, преимуществ |
| 2x2 / 2x3 grid | Сравнения, категории |
| Half-bleed image + overlay | Hero/impact слайды |
| Large stat callout (44-50pt число) | Метрики, данные |
| Timeline / process flow | Последовательности, roadmaps |
| Quote slide | Цитаты, ключевые высказывания |

**Правило:** каждый слайд должен иметь визуальный элемент — иконку, фигуру, картинку или диаграмму. Plain bullets на белом фоне = недопустимо.

## Параметры картинок

| Параметр | Значение |
|----------|----------|
| Формат | 3:2 (горизонтальная) или 1:1 (квадрат) |
| Размеры | 1200×800px или 900×600px / 800×800px или 1000×1000px |
| Обводка | 1px, Dark Teal `#008B7F` |
| Тень | Dark Teal 25%, угол 45°, расстояние 7px, размытие 5px |
| Скругление | 20-40px |

### Иконки

- Цвет: teal (`#008B7F`) или bright teal (`#00B4A6`)
- Стиль: teal круг с белой иконкой, или без фона с teal иконкой

- **Teal stripe:** тонкая полоска вверху каждого слайда (на всю ширину, ~1mm)

- **Pink dot** в "svaib." — единственное структурное использование розового
- **Без accent underlines** под заголовками

## Art Director Prompt

Промпт для генерации иллюстраций — хранится в инструкции арт-директора (отдельный контекст).

Стиль: современная иллюстрация, Bauhaus-структура с Matisse/Kandinsky-игривостью. Flat design, геометрические формы, ограниченная палитра (teal, dark blue, yellow, pink = только акцент).
