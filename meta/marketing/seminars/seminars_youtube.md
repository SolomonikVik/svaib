# Семинар → YouTube: чеклист публикации

## Кратко

Повторяемый процесс: от Zoom-записи до опубликованного видео на YouTube. Две части: одноразовая подготовка (делается один раз) и чеклист на каждый семинар (~15 минут).

Дополняет seminars_setup.md (подготовка ДО семинара). Этот файл — всё что ПОСЛЕ.

## Граница "гигиена vs монтаж"

Допускается гигиеническая обработка — но с жёсткой границей (см. правило №3 в [seminars_ai_brain.md](seminars_ai_brain.md)).

**Гигиена (можно):**
- Обрезать "мёртвое" начало (до старта контента) и конец (после прощания)
- Вырезать технические паузы >30 сек ("подождём, пока все подключатся", проблемы со звуком)
- Auto-enhance звука (одна кнопка в iMovie)
- Fade in/out (1 сек)

**Монтаж (нельзя — защита от перфекционизма):**
- Переставлять куски местами
- Добавлять заставки, интро, музыку
- Цветокоррекция
- Вставлять дополнительные слайды или графику
- Дублировать или повторять фрагменты

Тест: если действие занимает >1 минуту на один разрез — это монтаж, не гигиена.

## Чеклист на каждый семинар

### Этап 1: Подготовка материалов (Виктор + AI)

**AI делает (по транскрипту):**
- [ ] Таймкоды (YouTube chapters формат: `00:00 Тема`)
- [ ] Название видео (рабочее, без кликбейта)
- [ ] Описание (2-3 предложения + ссылки: Telegram, сайт)
- [ ] Рекомендации что вырезать (с таймкодами, по транскрипту)

**Виктор проверяет:**
- [ ] Просмотреть рекомендации — решить что вырезать

### Этап 2: Обработка видео в iMovie (~10 мин)

- [ ] Скачать запись из Zoom Cloud (zoom.us → Recordings → Cloud Recordings → Download)
- [ ] Импортировать в iMovie (перетащить файл на таймлайн)
- [ ] Обрезать начало и конец (перетаскивание краёв клипа)
- [ ] Вырезать помеченные куски (если решил)
- [ ] Громкость: выделить клип → кнопка "Volume" над таймлайном → нажать **Auto** (выравнивает громкость)
- [ ] Шумоподавление: кнопка "Noise Reduction and Equalizer" → включить шумоподавление + пресет **Voice Enhance**
- [ ] Fade in/out: потянуть маленький кружок (fade handle) в начале и конце клипа → ~1 сек
- [ ] Экспорт: File → Share → File → Resolution **1080p**, Quality **High** → сохранить .mp4

### Этап 3: Публикация (~5 мин)

- [ ] Залить на YouTube → канал "Соломоник Виктор" → плейлист "Путь к личному Second AI Brain"
- [ ] Вставить: название, описание, таймкоды (из подготовленного AI)
- [ ] Обложка: открыть шаблон → вписать тему → загрузить
- [ ] Visibility: Public (или Unlisted — если хочешь сначала проверить)

### Этап 4: Дистрибуция

- [ ] Пост в Telegram: краткий итог семинара + ссылка на YouTube
- [ ] Обновить лог в seminars_ai_brain.md (дата, тема, участники, ссылка на запись)
