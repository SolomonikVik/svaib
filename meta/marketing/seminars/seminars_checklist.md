# Двухнедельный чеклист подготовки семинара

## Кратко

Операционный ритм: что делать каждую неделю между семинарами. Привязан к дням — задачи вытягиваются в план дня. Здесь же шаблоны постов. Техническая настройка Zoom (одноразовая) — в [seminars_setup.md](seminars_setup.md). Пост-продакшн YouTube — в [seminars_youtube.md](seminars_youtube.md).

## Связанные файлы

- [seminars_ai_brain.md](seminars_ai_brain.md) — программа, формат, лог семинаров
- [seminars_setup.md](seminars_setup.md) — настройки Zoom (одноразовые)
- [seminars_youtube.md](seminars_youtube.md) — пост-продакшн: запись → YouTube
- [Презентации (Google Drive)](https://drive.google.com/drive/u/0/folders/1IP-c4kWss_BmZhypqOs30nZFoy9Ph4w7) — материалы для демо

---

## Неделя 1 (чётная — после семинара)

Фокус: закрыть прошлый семинар, выдохнуть.

### Понедельник

- [ ] **Опубликовать запись** — полный чеклист в [seminars_youtube.md](seminars_youtube.md)
  - AI готовит материалы (таймкоды, описание, название) по транскрипту
  - iMovie: обрезка + звук (~10 мин)
  - YouTube: залить, оформить
  - Telegram: пост с итогом + ссылка на YouTube
- [ ] **Обновить лог** в [seminars_ai_brain.md](seminars_ai_brain.md) (дата, тема, участники, ссылка)

### Остаток недели

Семинар не трогаем. Клиенты, продукт, лаборатория.

---

## Неделя 2 (нечётная — подготовка к семинару)

Фокус: содержание → анонс → напоминания → проведение.

### Вторник — содержание (потолок: 2 часа)

- [ ] **Подготовить тезисы** — 5-7 ключевых мыслей, которые хочу донести


### Среда-четверг — анонс

- [ ] **Создать Zoom-ссылку + календарь:**
  1. Zoom (приложение) → Запланировать → тема "Путь к личному Second AI Brain"
  2. Перебрасывает в Google Calendar → сохранить
  3. Google Calendar → открыть событие → Пригласить с помощью ссылки → скопировать ссылку
- [ ] **Предпоготовка демо** — что буду показывать на экране / набросок презентации
- [ ] **Опубликовать анонс в Telegram** — пост с темой, датой, временем, ссылкой (шаблон ниже)


### Пятница

- [ ] **Финализировать демо** — что буду показывать на экране / презентация, если есть

### Суббота — проведение

- [ ] **Финальный прогон** — порядок, переходы, где возможны вопросы


**За 1 час:**
- [ ] Короткий reminder в Telegram (шаблон ниже)

**За 30 минут - ТЕХ. ПОДГОТОВКА:**
- [ ] Проверить микрофон (записать тестовое 1-мин видео в Zoom)
- [ ] Поставить свет
- [ ] Закрыть все загрузки/торренты

**За 5 минут:**
- [ ] Зайти в Zoom первым
- [ ] Убедиться что запись в облако включена (Record to the Cloud)

**После семинара:**
- [ ] Убедиться что запись сохранилась в Zoom Cloud

## Шаблоны постов

### Анонс (ср-чт, за 2-3 дня)

```
Привет!

В эту субботу — [какой по счёту] семинар нашего кружка «Путь к личному Second AI Brain».

Тема: [ТЕМА] — [короткое пояснение].

[Связь с прошлым семинаром: что разобрали → что теперь]

[2-3 предложения: о чём будем говорить, почему это важно, что участник унесёт с собой]

Что разберём:
— [пункт 1]
— [пункт 2]
— [пункт 3]
— [пункт 4]

Формат тот же: говорю базу, живой разговор, вопросы - ответы.

📅 Суббота, [ДД месяц]
🕘 09:00 МСК / 13:00 КРСК
📎 Добавить в календарь (там же Zoom-ссылка): [ССЫЛКА]

Запись будет — но живое участие ценнее.
```


### Reminder (сб, за 1 час)

```
Через час начинаем!

Семинар «[ТЕМА]»
Zoom: [ССЫЛКА]
```
