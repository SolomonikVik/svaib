# Brand — PDF-документы

## Кратко

Как делаем PDF-документы svaib: принципы, шрифты, элементы страницы, что работает и что нет. Выводы из создания первого коммерческого предложения (февраль 2026). Палитра и философия — в [brand.md](brand.md).

## Связанные файлы

- [brand.md](brand.md) — палитра, логотип, философия, универсальные правила

---

## Общий принцип

**Минимализм с акцентами.** Белый фон, чистая типографика, цвет — только для навигации и акцентов. Не раскрашивать всё подряд.

---

## Шрифты

| Элемент | Шрифт | Размер |
|---------|-------|--------|
| Логотип "svaib" | Sora | 18px |
| Заголовок (латиница) | Sora | 22-24px |
| Заголовок секции (кириллица) | Inter-Bold | 14px |
| Подзаголовок | Inter-SemiBold | 9.5-10px |
| Основной текст | Inter | 9-9.5px |
| Цены | Sora | 24-26px |
| Мелкий текст, подписи | Inter | 8-8.5px |
| Футер | Inter 7.5px + Sora 8px (логотип) |

**Важно:** Sora (Google Fonts) НЕ поддерживает кириллицу — все кириллические тексты только в Inter. Sora — только для латиницы: логотип, цены ($1,200), заголовок "Second AI Brain".

**Целевой шрифт** — Montserrat+Roboto (как на слайдах), но пока используем Inter из-за ограничений Sora с кириллицей.

## Палитра для PDF

Подмножество из [brand.md](brand.md), с ограничениями:

| Элемент | Цвет | HEX |
|---------|------|-----|
| Основной текст | Тёмно-серый | `#1A1A1A` |
| Вторичный текст | Серый | `#6B7280` |
| Третичный текст | Светло-серый | `#9CA3AF` |
| Акцент (структура) | Teal | `#00B4A6` |
| Акцент (логотип) | Pink | `#FF4D8D` |
| Фон | Белый | `#FFFFFF` |
| Разделители | Бордер | `#E5E7EB` |

**Правило:** teal для структуры, pink только в логотипе (точка). Максимум 2 цветовых акцента на страницу.

## Элементы страницы

### Что работает

**Header:**
- Тонкая teal-полоска вверху (2 мм) — маркер бренда
- Логотип svaib (Sora, teal) + розовая точка — компактный
- Дата справа вверху, мелким шрифтом

**Footer:**
- Логотип + svaib.com слева, имя автора справа
- Разделены тонкой линией

**Секции:**
- Teal-полоска слева от заголовка (3px × 5mm) — навигация без заливок
- Заголовки: Inter-Bold 14px, цвет `#1A1A1A`
- Между секциями — тонкий разделитель (`#E5E7EB`, 0.5px)

**Карточки:**
- Рамка без заливки фона
- Основной вариант — teal border (1.5px), альтернативный — gray border (1px)
- Цена крупным шрифтом (Sora 26px), детали мелким (Inter 9px)

**Таблицы:**
- Teal-линия сверху (1.5px), gray-линия снизу (0.5px)
- Без фоновых заливок строк

### Что НЕ работает

- **Цветные фоны** на карточках (teal, pink, light) — выглядит как "трэш"
- **Розовые элементы** — подчёркивания, badges, карточки → перебор
- **Градиенты** — не для документов
- **Фон страницы ≠ белый** — визуальный шум
- **Больше 2 цветовых акцентов** на страницу — перегруз

## Чеклист перед финализацией

- [ ] Все кириллические тексты в Inter (не Sora)
- [ ] Sora только для: логотипа, цен, латинских заголовков
- [ ] Розовый — только в логотипе (точка)
- [ ] Нет фоновых заливок (кроме белого)
- [ ] Карточки — рамка без заливки
- [ ] Максимум 2 цветовых акцента на страницу
- [ ] Текст не наезжает на элементы
- [ ] Визуальная проверка: конвертировать PDF в PNG

---

## Технические заметки (ReportLab)

- Sora скачивается как variable font (`Sora[wght].ttf`), ReportLab рендерит как Regular weight
- Для Bold/SemiBold нужны static instances (fonttools instancer) или отдельные файлы
- Inter-Bold / Inter-SemiBold / Inter-Regular — static TTF из GitHub release (v4.0)
- Sora используется только для латиницы, поэтому вес не критичен
