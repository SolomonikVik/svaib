# meta/product/

Продуктовая стратегия svaib: что строим, зачем, куда развиваем.

## Отличие от смежных папок

| Папка | Фокус |
|-------|-------|
| **meta/product/** | ЧТО строим и ЗАЧЕМ (стратегия, видение, идеи) |
| **framework/** | КАК устроен продукт (артефакт для клиента: онтология, методология) |
| **dev/** | КАК реализуем технически (код, инфра, деплой) |

## Файлы

| Файл | Назначение |
|------|-----------|
| product_vision.md | Видение продукта, бизнес-модель, эволюция |
| model.md | Модель продукта для презентации клиентам (компоненты, подписка, результаты) |

## Связи

- meta/management/01_vision.md → секция "Продукт" ссылается сюда
- Продуктовые решения отсюда → влияют на framework/ (что появляется в онтологии)
- framework/ → влияет на dev/ (что нужно закодить)
