# meta/

> Часть проекта svaib. Полная структура: [../README.md](../README.md)

## Назначение

Стратегия проекта и продукта. Отвечает на вопросы ЧТО, ЗАЧЕМ, КТО, КОГДА. Включает описание проекта, продукта, маркетинг, управление.

## Структура

```
meta/
├── management/          — Стратегия, OKR, прогресс, команда
├── marketing/           — Бренд, каналы, контент
├── product/             — Позиционирование продукта
├── my_profile_viktor.md — Профиль Виктора
└── z_archive/           — Архив устаревших документов
```

## Подпапки

| Папка | Что внутри | Когда читать |
|-------|-----------|--------------|
| [management/](management/) | Vision, OKR, план, прогресс, решения, команда | Стратегия, планирование |
| [marketing/](marketing/) | Бренд, дизайн-гайды, каналы, стратегия | Контент, брендинг |
| [product/](product/) | Плагин, позиционирование, landscape | Продукт |

## Связанные папки

- **framework/** — Ядро продукта Second AI Brain (онтология, методология, плагин)
- **dev/** — Разработка и реализация (КАК строить продукт)
- **lab/** — AI-лаборатория (принципы, процессы, правила создания помощников)
