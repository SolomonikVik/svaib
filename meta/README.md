# meta/

> Часть проекта svaib. Полная структура: [../README.md](../README.md)

## Назначение

Стратегия проекта svaib. Отвечает на вопросы ЧТО, ЗАЧЕМ, КТО, КОГДА. Включает vision, OKR, управление, маркетинг. **Продуктовое видение и архитектура Second AI Brain живут в [../framework/](../framework/)** — отдельной папки «продукт» в meta нет.

## Структура

```
meta/
├── management/          — Стратегия, OKR, план, прогресс, решения
├── marketing/           — Бренд, каналы, контент
├── my_profile_viktor.md — Профиль Виктора
└── z_archive/           — Архив устаревших документов
```
