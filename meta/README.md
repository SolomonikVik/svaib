# meta/

> Часть проекта svaib. Полная структура: [../README.md](../README.md)

## Назначение

Стратегия проекта и продукта. Отвечает на вопросы ЧТО, ЗАЧЕМ, КТО, КОГДА. Включает описание проекта, продукта, маркетинг, команду, исследования.

## Структура

```
meta/
├── meta_context/  — Ключевые документы проекта
├── prompts/       — Промпты для AI-агентов
├── research/      — Исследования рынка
└── z_archive/     — Архив устаревших документов
```
